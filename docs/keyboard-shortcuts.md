# Vendor Dashboard Keyboard Shortcuts

## Overview
The vendor dashboard supports keyboard shortcuts for power users to navigate and perform actions more efficiently.

## Global Shortcuts

### Navigation
| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl/Cmd + K` | Open Search | Quickly search for products, orders, or customers |
| `Esc` | Close Modal/Dialog | Close any open modal, dialog, or sheet |
| `?` | Show Help | Display keyboard shortcuts help modal |

### Actions
| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl/Cmd + N` | Add New Product | Open the add product wizard |
| `Ctrl/Cmd + S` | Save | Save current form (when in edit mode) |

### Dashboard Navigation
| Shortcut | Action | Description |
|----------|--------|-------------|
| `Tab` | Next Field | Move to next form field or interactive element |
| `Shift + Tab` | Previous Field | Move to previous form field |
| `Enter` | Activate/Submit | Activate button or submit form |
| `Space` | Toggle | Toggle checkboxes or switches |

## Page-Specific Shortcuts

### Products Page
| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl/Cmd + A` | Select All | Select all visible products |
| `Ctrl/Cmd + D` | Deselect All | Clear product selection |
| `Delete` | Delete Selected | Delete selected products (with confirmation) |
| `E` | Edit | Edit selected product (single selection) |

### Orders Page
| Shortcut | Action | Description |
|----------|--------|-------------|
| `F` | Filter | Open filter options |
| `R` | Refresh | Refresh order list |
| `P` | Print | Print selected order |

## Form Navigation

### Add/Edit Product Forms
| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl/Cmd + Enter` | Submit Form | Save and submit product form |
| `Esc` | Cancel | Cancel form and return to list |
| `Tab` | Next Field | Move to next form field |
| `Shift + Tab` | Previous Field | Move to previous form field |

### Image Upload
| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl/Cmd + V` | Paste Image | Paste image from clipboard |
| `Delete` | Remove Image | Remove selected/focused image |
| Arrow Keys | Navigate Images | Navigate through uploaded images |

## Modal & Dialog Navigation
| Shortcut | Action | Description |
|----------|--------|-------------|
| `Esc` | Close | Close current modal or dialog |
| `Enter` | Confirm | Confirm action in dialog |
| `Tab` | Cycle Focus | Move focus through dialog elements |

## Accessibility Features

### Screen Reader Support
- All interactive elements have proper ARIA labels
- Form fields announce errors and hints
- Navigation landmarks for page structure
- Live regions for notifications and updates

### Focus Management
- Focus is trapped within modals and dialogs
- Focus returns to trigger element when closing
- Visible focus indicators on all interactive elements
- Skip links for keyboard navigation

## Tips for Efficient Use

1. **Learn the Core Shortcuts First**
   - Start with `Ctrl/Cmd + K` for search
   - Master `Ctrl/Cmd + N` for adding products
   - Use `Esc` to quickly close dialogs

2. **Use Tab Navigation**
   - Tab through forms instead of using mouse
   - Shift + Tab to go backwards
   - Enter to submit when on a button

3. **Combine with Mouse**
   - Use keyboard for quick actions
   - Use mouse for complex selections
   - Use keyboard for final confirmation

4. **Enable Browser Extensions**
   - Consider using Vimium or similar extensions
   - Configure custom shortcuts for frequently used pages

## Troubleshooting

### Shortcuts Not Working?

1. **Check Browser Conflicts**
   - Some browsers reserve certain shortcuts
   - Try disabling browser extensions
   - Check browser shortcut settings

2. **Focus Issues**
   - Ensure focus is in the correct area
   - Click on the page before using shortcuts
   - Check if modal/dialog is open

3. **Form-Specific Shortcuts**
   - Some shortcuts only work in specific contexts
   - Ensure you're on the correct page/form
   - Check if the feature is enabled

### Reporting Issues

If keyboard shortcuts aren't working:
1. Note which shortcut is failing
2. Specify browser and OS version
3. Describe steps to reproduce
4. Report to support team
